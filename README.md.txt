# 📚 Library Management System

A desktop-based Library Management System developed using **Java Swing**, **JDBC**, and **MySQL**. This application provides an easy-to-use graphical interface for managing books, students, and library transactions efficiently.

---

## ✨ Features

- 🔐 User Login Authentication
- 📖 Add, Update and Delete Books
- 👨‍🎓 Add, Update and Delete Students
- 📕 Issue Books
- 📗 Return Books
- 📊 Dashboard with Statistics
- 🔍 Search Books and Students
- 🖥️ User-friendly Java Swing GUI

---

## 🛠️ Technologies Used

- Java
- Java Swing
- JDBC
- MySQL
- NetBeans IDE

## ▶️ How to Run

1. Clone this repository.
2. Open the project in NetBeans IDE.
3. Create the required MySQL database.
4. Update the database username and password in the source code.
5. Run the project.
